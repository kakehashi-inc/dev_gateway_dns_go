## テーブル定義

---

### proxy_rules

プロキシルール定義テーブル。

| カラム名 | 型 | 制約 | 説明 |
|---|---|---|---|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自動採番 |
| hostname | TEXT | NOT NULL, UNIQUE | ホスト名 (例: foo.local) |
| backend_protocol | TEXT | NOT NULL, DEFAULT 'http' | バックエンドへの転送プロトコル ("http" or "https") |
| backend_ip | TEXT | NULL可 | 転送先IP (NULL=自動, 値あり=手動指定) |
| backend_port | INTEGER | NOT NULL | 転送先ポート (例: 8080) |
| enabled | BOOLEAN | NOT NULL, DEFAULT 1 | 有効/無効フラグ |
| created_at | DATETIME | NOT NULL | 作成日時 |
| updated_at | DATETIME | NOT NULL | 更新日時 |

リレーション: なし

---

### dns_records

手動DNSレコード管理テーブル。自動レコード（プロキシ連動Aレコード）はメモリ上で管理されるためこのテーブルには含まれない。

| カラム名 | 型 | 制約 | 説明 |
|---|---|---|---|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自動採番 |
| name | TEXT | NOT NULL | FQDN (例: mail.example.local) |
| type | TEXT | NOT NULL | レコードタイプ (A, AAAA, CNAME, MX, TXT等) |
| value | TEXT | NOT NULL | レコード値 |
| ttl | INTEGER | NOT NULL, DEFAULT 300 | TTL秒数 |
| priority | INTEGER | NULL可 | 優先度 (MX, SRV用) |
| weight | INTEGER | NULL可 | 重み (SRV用) |
| port | INTEGER | NULL可 | ポート (SRV用) |
| created_at | DATETIME | NOT NULL | 作成日時 |
| updated_at | DATETIME | NOT NULL | 更新日時 |

リレーション: なし

---

### ca_certificate

ルートCA証明書テーブル (1行のみ)。

| カラム名 | 型 | 制約 | 説明 |
|---|---|---|---|
| id | INTEGER | PRIMARY KEY (常に1) | 固定ID |
| cert_pem | BLOB | NOT NULL | CA証明書PEM |
| key_pem | BLOB | NOT NULL | CA秘密鍵PEM |
| expires_at | DATETIME | NOT NULL | 有効期限 |
| created_at | DATETIME | NOT NULL | 作成日時 |

リレーション: なし

---

### host_certificates

ホスト別SSL証明書テーブル。

| カラム名 | 型 | 制約 | 説明 |
|---|---|---|---|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自動採番 |
| hostname | TEXT | NOT NULL, UNIQUE | 対象ホスト名 |
| cert_pem | BLOB | NOT NULL | 証明書PEM |
| key_pem | BLOB | NOT NULL | 秘密鍵PEM |
| expires_at | DATETIME | NOT NULL | 有効期限 |
| created_at | DATETIME | NOT NULL | 作成日時 |

リレーション: なし (proxy_rulesのhostnameと論理的に関連)

---

### settings

システム設定テーブル (key-valueストア)。

| カラム名 | 型 | 制約 | 説明 |
|---|---|---|---|
| key | TEXT | PRIMARY KEY | 設定キー |
| value | TEXT | NOT NULL | 設定値 (JSON形式) |
| updated_at | DATETIME | NOT NULL | 更新日時 |

リレーション: なし

主な設定キー:

| key | 内容 | デフォルト値 |
|---|---|---|
| http_port | HTTP受付ポート | 80 |
| https_port | HTTPS受付ポート | 443 |
| dns_port | DNS受付ポート | 53 |
| proxy_port | フォワードプロキシポート | 8888 |
| admin_port | 管理UIポート | 9090 |
| listen_addresses | LISTENアドレスリスト (JSON配列) | ["0.0.0.0"] |
| upstream_dns_fallback | フォールバック上流DNS (JSON配列) | ["8.8.8.8","1.1.1.1"] |
| dns_query_history_size | DNSクエリ履歴保持件数 | 1000 |
| log_level | ログレベル (JSON文字列) | "info" |

---

### access_logs

HTTPアクセスログテーブル。

| カラム名 | 型 | 制約 | 説明 |
|---|---|---|---|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自動採番 |
| timestamp | DATETIME | NOT NULL | アクセス日時 |
| source | TEXT | NOT NULL | アクセス経路 ("reverse" or "forward") |
| client_ip | TEXT | NOT NULL | クライアントIP |
| hostname | TEXT | NOT NULL | リクエストホスト名 |
| method | TEXT | NOT NULL | HTTPメソッド |
| path | TEXT | NOT NULL | リクエストパス |
| status_code | INTEGER | NOT NULL | レスポンスステータスコード |
| response_time_ms | INTEGER | NOT NULL | 応答時間 (ミリ秒) |
| backend | TEXT | NOT NULL | 転送先 (ip:port) |

リレーション: なし
